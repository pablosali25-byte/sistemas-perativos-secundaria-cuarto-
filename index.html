// 01 · Definición
¿Qué es un Sistema
Operativo?
Un Sistema Operativo (SO) es el software fundamental que actúa como intermediario entre el hardware de la computadora y los programas del usuario. Sin él, la máquina no puede funcionar.

// 03 · Historia
Breve historia
de los SO
1950s
Sin SO. Los programas se cargaban con tarjetas perforadas, uno a la vez.
1960s
Primeros SO. IBM OS/360 introduce la multitarea: varias tareas a la vez.
1970s
Unix nace en Bell Labs. Influyó en casi todos los SO modernos.
1985
Windows 1.0 y Macintosh OS: llega la interfaz gráfica al público masivo.
1991
Linus Torvalds crea el núcleo Linux. Nace el SO libre más importante de la historia.
2000s+
Expansión a celulares: iOS (2007) y Android (2008). Los SO llegan a todos los bolsillos.
// 04 · Tipos
Tipos de
Sistemas Operativos
📜
Por licencia
Propietario — código cerrado (Windows, macOS)
Código abierto — código visible (Linux, Ubuntu)
🖱️
Por interfaz
GUI — ventanas e íconos (Windows, macOS)
CLI — solo texto y comandos (MS-DOS, Alpine)
👥
Por usuarios
Mono-usuario — un usuario a la vez
Multi-usuario — varios usuarios simultáneos
⚡
Por tareas
Monotarea — una tarea a la vez
Multitarea — varias tareas al mismo tiempo

// 06 · Software Libre
¿Quién definió
el Software Libre?
RMS

Richard Stallman

Nacido en 1953 · Nueva York
Programador, activista y filósofo

Fundador GNU · 1983
"El software libre es una cuestión de libertad, no de precio." — Richard Stallman, 1985
1983 Lanza el Proyecto GNU para crear un SO completamente libre.
1985 Funda la Free Software Foundation (FSF) y publica el Manifiesto GNU.
1989 Crea la licencia GPL — base legal del software libre.
1991 Torvalds crea Linux, que se combina con GNU para formar GNU/Linux.
// 07 · Las 4 Libertades
Las 4 Libertades
del Software Libre
Definidas por R. Stallman y la FSF · desde 1985
0
Libertad de USAR
Ejecutar el programa para cualquier propósito, sin restricciones.
Ej: Usás Linux en casa, en la escuela o en una empresa sin pedir permiso.
1
Libertad de ESTUDIAR
Acceder al código fuente y entender cómo funciona el programa.
Ej: Cualquiera puede leer y aprender del código de Ubuntu.
2
Libertad de DISTRIBUIR
Copiar y compartir el programa libremente con quien lo necesite.
Ej: Podés copiar Linux en un pendrive y dárselo a un amigo.
3
Libertad de MEJORAR
Modificar el programa y publicar esas mejoras para toda la comunidad.
Ej: Ubuntu nació de mejoras hechas sobre Debian por Canonical.
⚠️ "Libre" no significa gratuito — significa libre como en libertad de expresión, no como en cerveza gratis. (Stallman, 1985)
